## How to Take Screenshots:

### Windows:
- Press `Windows + Shift + S` to open Snipping Tool
- Select area and save

### Mac:
- Press `Cmd + Shift + 4` to take screenshot
- Click and drag to select area

### Browser DevTools:
- Press `F12` to open DevTools
- Press `Ctrl + Shift + M` (Windows) or `Cmd + Shift + M` (Mac) for device toolbar
- Take screenshots of different screen sizes

## Tips:
- Use consistent browser window size
- Clear browser cache for clean screenshots
- Show realistic data in cart
- Capture both desktop and mobile views
